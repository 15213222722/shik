# Shik
### shik-cloud
- spring-cloud包含注册中心，服务，消费，config server及zuul的路由拦截等demo
### shik-RA-server
- 集群spring cloud eureka
### shik-zuul-gateway
- 集群spring cloud zuul 
### shik-config-server
- 集群spring cloud config
- 集成spring cloud bus, 实现公共配置的热更新并向eureka实例发送通知
### shik-common
- shik-common为shik项目公共部分，提供pom依赖、config-repo和util等
